# `install.ps1`

## Описание
Устанавливает Ollama как службу Windows с автозапуском.

## Параметры
- **`$ServiceUser`** (опционально)  
  Имя пользователя службы. По умолчанию: `OllamaService`

## Что делает
1. Запрашивает пароль (с подтверждением)
2. Создаёт пользователя `OllamaService`
3. Копирует `ollama.exe` в `bin/`
4. Ищет или запрашивает путь к моделям
5. Настраивает права (ACL)
6. Устанавливает службу через `nssm`
7. Проверяет API

## Требования
- Права администратора
- Установленный Ollama (обычная версия)

## Пример
```powershell
PowerShell -ExecutionPolicy Bypass -File ".\scripts\install.ps1"
