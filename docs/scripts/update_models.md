# `update_models.ps1`

## Описание
Интерактивный скрипт для обновления установленных моделей Ollama.

Автоматически:
- Определяет путь к моделям из `.env`
- Проверяет, какие модели доступны для обновления
- Позволяет выбрать модели для обновления
- Загружает новые версии

## Параметры
Нет. Скрипт работает автономно.

## Что делает
1. **Читает `OLLAMA_MODELS` из `config/.env`**
2. **Проверяет доступ к папке с моделями**
3. **Получает список установленных моделей** через `ollama list`
4. **Проверяет каждую модель на наличие обновлений** (`ollama show --modelfile`)
5. **Показывает список моделей с возможностью выбора:**
   - По номерам (например: `1,3`)
   - Или `all` для всех
6. **Запрашивает подтверждение**
7. **Обновляет выбранные модели** через `ollama pull`

## Логирование
- Логи сохраняются в: `logs/model_update_YYYY-MM-DD.log`
- Временные файлы — в `%TEMP%\ollama-update\`

## Требования
- Ollama должен быть установлен и работать
- Служба `OllamaService` должна быть запущена
- Доступ в интернет

## Пример
```powershell
PowerShell -ExecutionPolicy Bypass -File ".\scripts\update_models.ps1"
Примечание
Скрипт не перезапускает службу — обновлённые модели станут доступны при следующем вызове ollama run
