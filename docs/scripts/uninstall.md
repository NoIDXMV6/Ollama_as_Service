# `uninstall.ps1`

## Описание
Полностью удаляет службу Ollama, включая:
- Остановку и удаление службы
- Удаление запланированных задач
- Удаление правила брандмауэра
- Удаление локального пользователя
- Удаление всех файлов проекта

## Параметры
Нет. Скрипт работает автономно.

## Что делает
1. **Останавливает службу `OllamaService`** через `nssm`
2. **Удаляет службу**
3. **Удаляет запланированные задачи:**
   - `Ollama - Daily Restart`
   - `Ollama - Update Models`
4. **Удаляет правило брандмауэра:**  
   `Ollama - TCP 11434 Inbound`
5. **Удаляет пользователя `OllamaService`**
6. **Удаляет всю папку проекта** (`C:\Program Files\OllamaService` или аналогичную)

## Требования
- Права администратора
- Служба должна быть установлена через `install.ps1`

## Пример
```powershell
PowerShell -ExecutionPolicy Bypass -File ".\scripts\uninstall.ps1"
